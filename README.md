# Cali Coach (demo)
Live handstand alignment + push-up form scoring in the browser. Pose estimation runs fully on-device (MediaPipe); no video leaves the phone. Part of the CaliUnity ecosystem.
Open https://raw.githack.com/ollie-cali/cali-coach/main/index.html on a phone (Chrome/Safari), press Start camera, point it side-on at an athlete.
